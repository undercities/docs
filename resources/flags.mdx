### **What are Flags?**
Flags are optional parameters you can use to change how a message or command behaves.

### **What flags can I use?**
<Info>
  Reminder that these don't work on every command.
</Info>

- `--not_strict` - Make it respond to the trigger if it's in a sentence, instead of just the word alone
- `--reply` - Make it reply to the trigger instead of sending a straight up message.
- `--self_destruct (after time)` - Deletes the response after a time in seconds.
- `--delete_trigger` - Deletes the message it responded or replied to.

### **Examples of Flags in a command**

- `autoresponder add cal, the goat --self_destruct 5`
- `react add cal, 🍃 --not_strict`
