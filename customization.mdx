### **What is Customization?**

Customize the bot's appearance in your server, such as profile picture, about me, banner, and nickname to make rive stand out.

### **How do I customize the bot?**

<AccordionGroup>
  <Accordion title="Change the bot's profile picture">
    Run `customize avatar` to change the bot's avatar.

    <CodeGroup>

    ```javascript Syntax
    ,customize avatar (attach image or url)
    ```

    ```javascript Example
    ,customize avatar https://cdn.discordapp.com/1/2/3
    ```

    </CodeGroup>
  </Accordion>

  <Accordion title="Change the bot's nickname">
    Run `customize nickname` to change the bot's nickname.

    <CodeGroup>

    ```javascript Syntax
    ,customize nickname (new name)
    ```

    ```javascript Example
    ,customize nickname best bot
    ```

    </CodeGroup>
  </Accordion>

  <Accordion title="Change the bot's banner">
    Run `customize banner` to change the bot's banner.

    <CodeGroup>

    ```javascript Syntax
    ,customize banner (attach image or url)
    ```

    ```javascript Example
    ,customize avatar https://cdn.discordapp.com/1/2/3
    ```

    </CodeGroup>
  </Accordion>
</AccordionGroup>

### **How do I reset the customization?**

- `customize reset` - Reset all of the bot's customization
- `customize reset avatar` - Reset the bot's avatar customization
- `customize reset nickname` - Reset the bot's nickname customization
- `customize reset banner` - Reset the bot's banner customization
